Inverse T

Programming invT into a Ti 83 or Ti 83+ Calculator

PROGRAMM: INVT
:INPUT "AREA LEFT:", A
:INPUT "DF:", D
:1->S
:If A<0.5
:Then
:-1->S
:End
:abs(2*A - 1)->A
:TInterval 0, sqrt(D+1), (D+1), A
:Disp upper*S
:

T-score of Single Population

PROGRAMM: TSCORE1
INPUT "SAMPLE MEAN", X
INPUT "POPULATION MEAN", U
INPUT "SAMPLE SIZE", N
INPUT "SAMPLE SD", S
(X-U)/(S/sqrt(N))

  • Select A
  • Specify rows and columns for the matrix 2 x 3
  • Input given numbers
  1. Test with Chi-squared Test
  • Press STAT, select TESTS
  • Press ALPHA, then press C to select X2\Chi^2X2-Test
  1. Select Calculate or Draw to see result
  2. To check expected value
  • Press 2nd, x-1/MATRIX, select EDIT on screen
