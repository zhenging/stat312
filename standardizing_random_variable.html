Standardizing Random Variables

The idea of standardizing a random variable can be applied to other families as well, by substracting the expected value (mean) and dividing by the standard deviation.

Y−E[Y]VAR[Y]=standardized version of Y\begin{aligned} \frac{Y -E[Y]}{\sqrt{\textrm{VAR}[Y]}} = \text{standardized version of } Y \end{aligned} VAR[Y]​Y−E[Y]​=standardized version of Y​
