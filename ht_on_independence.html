HT on Independence

  • We have two variable and we want to test whether they are independent or not.
  • H0H_0H0​: Variable A is independent of variable B.
  • HAH_AHA​: Variable A **is NOT **independent of variable B.
  • Independence is commutative, os it does not matter which one is listed first.

HT on Independence for 2 Categorical Vars

  1. Construct a contingency table of observed values
  2. Calculate OijO{ij}Oij and EijE{ij}Eij for all cells
  • OijO{ij}Oij = observed count in cell (i, j)
  • EijE{ij}Eij = expected count in cell (i, j)
  • Assuming H0H_0H0​ is true, the variables are inpdependent
  • Eij=p^i∗p^j∗n=(count in row i) * (count in column j)nE{ij} = \hat p_i * \hat p_j * n = \frac{\text{(count in row i) * \text{(count in column j)}}}{n}Eij=p^​i​∗p^​j​∗n=n(count in row i) * (count in column j)​
  • Large difference between OijO{ij}Oij and EijE{ij}Eij given evidence to reject H0H_0H0​.
